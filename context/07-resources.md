# 07 — Resources for PM Builders

A curated list of resources for product managers learning to build with AI. Quality over quantity — every link here earns its spot.

---

## 1. Official Documentation

Start here. These are your reference manuals.

- **[Claude Code Docs](https://code.claude.com/docs/en/overview)** — Anthropic's agentic coding tool. Terminal-native, codebase-aware.
- **[Cursor Docs](https://cursor.com/docs)** — AI-first code editor. Best quickstart for visual learners.
- **[GitHub Docs: Git Basics](https://docs.github.com/en/get-started/getting-started-with-git)** — Version control fundamentals. Non-negotiable foundation.
- **[Vercel v0 Docs](https://v0.dev/docs)** — Prompt-to-UI generation. Great for PMs prototyping interfaces.

## 2. Learning Platforms

### Courses for PMs Who Build

- **[Vibe Coding for Product Managers](https://jackiebavaro.substack.com/p/vibe-coding-for-product-managers)** by Jackie Bavaro — Free. The best starting point for PMs, from the co-author of *Cracking the PM Interview*.
- **[Vibe Coding 101: Kickstart 2026](https://maven.com/p/3ee8d0/vibe-coding-101-kickstart-2026)** (Maven) — Live cohort. Idea to working app in 4 weeks using natural language.
- **[Vibe Coding for Non-Technical Founders](https://maven.com/drewfalkman/vibe-coding-for-non-technical-founders)** (Maven) — Data-enabled AI apps. Good for PMs and designers.
- **[Udemy: Lovable AI Complete Guide](https://www.udemy.com/course/lovable-ai-complete-guide-for-vibe-coding/)** — Build and publish a real app with Lovable + Supabase.
- **[HelloPM: Free AI PM Course](https://hellopm.co/free-ai-pm-course/)** — Free. Covers how to spot AI opportunities and work with engineering teams.

### Skills Discovery

- **[Skills.sh](https://skills.sh/)** — Open directory of reusable AI agent skills (by Vercel). Browse what agents can do, install skills as markdown files. Install with `npx skills add <owner>/<repo>`.

## 5. Books & Long Reads

Short list. Read in this order if you're starting from zero.

1. **The AI Product Manager's Handbook** — Irene Bratsis. End-to-end guide: from understanding ML to shipping AI products.
2. **Inspired** — Marty Cagan. Not AI-specific, but the product thinking foundation you need before building anything.
3. **The Vibe Coding Imperative for Product Managers** — [ACM article](https://cacm.acm.org/blogcacm/the-vibe-coding-imperative-for-product-managers/). Free. Frames why this shift matters.
4. **AI Engineering** — Chip Huyen. For when you're ready to go deeper on how LLM-powered products actually work.

## 6. Tools Reference

Quick comparison of AI coding tools relevant to PM builders (as of Feb 2026).

| Tool | Price/mo | Best For | Learning Curve (1-5) |
|---|---|---|---|
| **[Cursor](https://cursor.com)** | $20 (Pro) | Full codebase editing, complex projects | 4 |
| **[Claude Code](https://code.claude.com)** | Usage-based* | Terminal power users, agentic workflows | 4 |
| **[Windsurf](https://windsurf.com)** | $15 | Budget-friendly IDE, predictable pricing | 3 |
| **[v0](https://v0.dev)** | Free / $20 (Pro) | UI prototyping from prompts | 2 |
| **[Lovable](https://lovable.dev)** | $25 (Pro) | Non-technical builders, full-stack MVPs | 1 |
| **[Bolt.new](https://bolt.new)** | ~$20 (token-based) | Fast prototyping, instant deploys | 2 |
| **[Replit](https://replit.com)** | Free / $25 (Core) | Learning to code, collaborative editing | 2 |

*Claude Code: Free via Claude Pro ($20/mo) with limits; Max plans at $100-200/mo for heavy use.

**How to choose:** If you've never coded, start with **Lovable** or **v0**. If you want to learn real development, start with **Cursor**. If you're terminal-comfortable, **Claude Code** is the most capable agentic tool.

## 7. Prompt Libraries

Don't start from scratch. Steal from the best.

- **[PM Prompt Library](https://www.news.aakashg.com/p/pm-prompt-library)** — 82 tested prompts for PRDs, strategy, research. Works with Claude and ChatGPT.
- **[Product Manager Prompts](https://github.com/deanpeters/product-manager-prompts)** (GitHub) — Open-source. Uses PM frameworks like JTBD and Value Proposition Canvas.
- **[Productboard AI Prompt Templates](https://www.productboard.com/blog/6-ai-prompt-templates-for-product-managers/)** — 6 templates for common PM workflows.
- **[Anthropic Prompt Library](https://docs.anthropic.com/en/prompt-library)** — Official Claude prompts. Good foundation for building your own.
